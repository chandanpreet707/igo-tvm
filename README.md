# iGo - Ticket Vending Machine

iGoApplication is a JavaFX desktop application built with Spring Boot for managing TVM operations.

## Installation

### Prerequisites
* Java JDK 17+
* Maven
* Git

### Clone the repository
```bash
git clone https://github.com/chandanpreet707/igo-tvm.git
cd igo-tvm
```

### Build the project

```bash
mvn clean install
```

### Usage
#### Run with Maven
```bash
mvn javafx:run
```
